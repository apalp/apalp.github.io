# 模型思想
评价要考虑的方面：1.评价指标；2.指标权重
1. 判断矩阵（各种因素权重的倍数关系，仅考虑横向和纵向，两两比较）
2. 一致性检验（确定判断矩阵的偏差是否在可接受范围内）
3. 计算得分：算术平均法，几何平均法，特征值法（最多）

# 计算步骤
1. 确定目标层（要做的事），准则层（比较指标），方案层（实现目标的方案）
2. 数据处理，消除量纲（不同的单位带来的数值差异）：归一化（每个数据/所在列的和）
3. 层次分析法定权重

	- 列出判断矩阵（1，3稍重，5明显，7强烈，9极端，中间值线性插入？）横向指标/纵向指标--一致性矩阵（正互反矩阵与单元格关系）
	- 一致性检验（利用特征值）
		- - 计算判断矩阵的最大特征值所参与计算的CR（一致性比例）是否满足一致性检验
		- 不满足重新调整判断矩阵的值（以第一行为例）
	- 满足，计算权重（算术平均法，集合平均法，特征值）：
				算术平均法：按列归一化，按行求均值
				几何平均法：按行相乘求n次根号，按列归一化
				特征值法：最大特征值所对应的特征向量，对其进行归一化
4. 计算得分：指标的数据×权重的数据
