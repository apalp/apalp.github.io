matlab不要求事前对所使用的变量进行声明，也不要求指定变量的类型（matlab自动根据所赋给变量的值和对变量的操作对其进行赋值）
变量名命名规范：
1. 字母开头，允许数字字母下划线
2. 区分大小写
3. 长度不超过63个字符
4. 变量名不能和关键字重名（iskeyword获得关键字）
变量的赋值
=（赋初值，复值）

|   特殊变量   | 描述（可重定义）                                    |
| :------: | ------------------------------------------- |
|   ans    | 系统默认用于保存运算结果变量名                             |
|    pi    | 圆周率                                         |
| inf/-inf | 无穷大/负无穷大                                    |
|   NaN    | 不定值或者缺失值                                    |
|   i和j    | 附属中的虚数单位                                    |
|   eps    | 浮点数的相对精度。可将其看作一个非常小的数2.22* 10^-16（2.22e-16） |
|          |                                             |

更改matlab的数值的显示方式
format long g（影响显示的方式）
对于某些计算结果（变量）并不需要在命令行显示，可以在它的后面加上分号；若要把这两行及以上代码语句行放在同一行也可以在两行之间加一个分号。
