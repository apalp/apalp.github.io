| 常见函数       | 描述                          |
| :--------- | :-------------------------- |
| abs()      | 求绝对值复数的模长                   |
| floor()    | 向下（负无穷）取整                   |
| fix（）      | 朝零取整/小数部分形式截断               |
| ceil（）     | 向上（正无穷）取整                   |
| round（）    | 最接近整数，0.5偏离0的方向；；           |
| round（X,N） | 将X在第N位上四舍五入N<0四舍五入到小数点左侧第N位 |
| mod（a，m）   | a/m得到的余数                    |
| sqrt（a）    | 对a开平方根；若a为负数则返回复数           |
| exp（a）     | e^a                         |
| log（a）     | 以自然常数e为底的对数                 |
| log2（a）    | 以2为底求对数                     |
| log10（a）   | 以10为底求对数                    |
